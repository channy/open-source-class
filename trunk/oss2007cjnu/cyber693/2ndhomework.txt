FireFox :
 
MS 인터넷 익스플로러 브라우저는 하나의 브라우저에서 하나의 사이트만 접속이 가능 하지만 FireFox
는 계속 윈도우창을 열면서 웹서핑을 할 수 있으므로 시간도 절약할 수 있고 매우 편리하다.
이외에 기능으로는 인터넷 익스플로러, 오페라, 네스케이프, 모질라 등의 웹 브라우저의 히스토리, 쿠
키, 즐겨찾기 등을 적용해 FireFox 에서 사용할 수 있으며 자체적으로 팝업창 억제 기능 내장, 구글/
아마존/야후 등의 검색툴바 지원, 강력한 기능의 히스토리 지원, 인터넷 익스플로러와 비교해 안전한 
보안기능, 테마 변경 기능, 자바 스크립트 콘솔 기능 등을 지원합니다.

이스가 눈길을 끄는 프로그램이다
